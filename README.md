## 🚀 Featured Project: Azure Data Pipeline (SARB Application)

### [🔗 Azure Data Pipeline for Financial Transaction Processing](https://github.com/MALADJI/sarb-azure-data-pipeline)

*Built specifically for the **SARB Grow-IT Internship 2027** application.*

An end-to-end, serverless, event-driven data pipeline deployed on **Microsoft Azure**.

**🛠 Tech Stack:**
- **Python 3.14** – Core programming language
- **Azure Functions** – Serverless, event-driven processing (Blob Trigger)
- **Azure Blob Storage** – Cloud data storage (Bronze & Gold layers)
- **Azure Managed Identity** – Password-less authentication (zero-trust security)
- **Azure SDK for Python** – Programmatic cloud interaction

**✅ Key Achievements:**
- Designed and implemented a **Medallion Architecture** data pipeline (Bronze → Silver → Gold)
- Configured **Managed Identity** for secure, password-less service-to-service authentication
- Deployed fully serverless solution to **Azure Cloud** (Consumption Plan)
- Automated processing of 500+ transaction records with real-time summary report generation
- Resolved Azure resource provider registration and region restriction issues

**📊 Pipeline Flow:**
CSV Upload → Blob Storage → Azure Function (Auto-trigger) → Summary Report

**🔐 Security:** IAM role-based access control, no hardcoded credentials in cloud environment
